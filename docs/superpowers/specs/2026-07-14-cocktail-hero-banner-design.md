# 칵테일 홈 배너 리뉴얼 설계

날짜: 2026-07-14

## 목표

칵테일 페이지(홈)의 히어로 배너를 참고 시안대로 교체한다.
코랄 배경 풀와이드 배너에 실제 앱 스크린샷(`main_phone.png`)과 실제 QR(`download_qr.png`)을 사용하고,
배너 좌상단에 흰색 로고, 우상단에 "앱 다운로드 하기" 텍스트를 배치한다.
기존에 모바일에서 숨겨지던 배너를 모바일에서도 표시한다.

## 결정 사항

- **페이지 구조**: 기존 NoticeBanner · SiteHeader는 그대로 유지. 배너는 그 아래에 위치하며
  배너 내부에 자체 로고 행(흰 로고 + 앱 다운로드 하기)을 가진다. (헤더와 로고 중복 노출 허용)
- **모바일 배치**: 데스크탑과 동일한 가로 구성(폰 왼쪽, 텍스트·QR 오른쪽)을 유지하되 전체 축소.
- **흰색 로고**: 별도 SVG 파일을 만들지 않고 `/logo.svg`에 `filter: brightness(0) invert(1)` 적용.

## 구현 범위

### `app/page/MasilengHome.jsx` — `AppHero` 함수 재작성

- 기존 폰 목업 DOM(2×3 카드 그리드)과 하드코딩 SVG QR(~200줄) 삭제.
- 새 구조:
  - `.cocktail-main-banner` (코랄 배경, 풀와이드)
    - `-inner` (max-width 1200px, 중앙 정렬)
      - `-top` 행: 흰 로고 이미지(좌) · "앱 다운로드 하기" 텍스트(우)
      - `-body` 행:
        - 좌: `main_phone.png` (하단이 배너 아래로 잘리는 구도)
        - 우: 타이틀 "내 손안의 레시피 / 마실랭"(흰색) · "지금 시작하세요" ·
          흰 라운드 박스 안 `download_qr.png` (라벨은 이미지에 포함되어 있어 별도 텍스트 없음)

### `app/css/banner.scss` — `.cocktail-main-banner` 섹션 재작성

- 배경: `var(--coral)`.
- 데스크탑: 내부 콘텐츠 `max-width: 1200px`.
- 모바일(≤768px):
  - 기존 `display: none` 목록에서 `.cocktail-main-banner` 제거 (슬라이딩·도전 배너는 유지).
  - 폰 이미지·타이틀·QR 등 전체 사이즈 축소.
  - 로고 행 하단 여백 최소화 — 로고/다운로드 텍스트가 콘텐츠 바로 위에 붙는다.
- 스타일 우선순위: 공통 변수(`--coral`, `--r-*`) → Tailwind → 인라인(단일 속성) → 커스텀 클래스.

## 에러 처리

- 이미지 로드 실패 시 alt 텍스트로 대체(브라우저 기본 동작). 별도 fallback 불필요.

## 검증

- 3000 포트에 떠 있는 기존 개발 서버 재사용.
- 데스크탑(1280px)·모바일(375px) 스크린샷으로 시안 대비 확인.
- 콘솔 에러 없음 확인.
